# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Initial release of OpenSem
- YAML configuration templates for TypeScript, Python, Full-stack, and Readonly modes
- Core memory templates for project documentation
- GitHub issue and PR templates
- Documentation for Claude Code integration

## [2.0.0] - 2025-02-04

### Added
- AI-native configuration system
- Pure YAML + AI intelligent generation approach
- Hybrid memory template structure
- Minimal documentation set
- Project renamed to OpenSem

### Changed
- Redesigned from script-based to AI-driven approach
- Simplified project structure
- Rebranded as OpenSem (Open Semantic Configuration System)

### Removed
- Script-based configuration system
- Complex template syntax
